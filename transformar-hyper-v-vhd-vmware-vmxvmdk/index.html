    Transformar Hyper-V (.vhd) a VMware (.vmx/.vmdk)

    Necesitaba una máquina virtual para crear mis aplicaciones de Biztalk. Buscando encuentro que desde Microsoft puedes bajar el Biztalk Server 2010 Developer Training Kit que incluye:
    - Microsoft Windows Server 2008 R2
    - Biztalk Server 2010
    - Visual Studio 2010
    - SQL Server 2008

    Todo configurado y listo para usar....pero, siempre hay peros, la imagen es para ejecutarla en un servidor Hyper-V y no disponemos de ninguno.

    Google me lleva a un blog donde explican como convertir de Hyper-V a VMware. Me pongo manos a la obra en un PC con Windows 7 de un compañero (igual que en el tutorial) pero no hay éxito, me da error siempre que intento convertir con el VMware Converter, tras probar varios workarounds me rindo y vuelvo a mi "querida" máquina con Windows XP 32bits a pesar de tener un procesador de 64 (a mi no me preguntéis)

    Me da por probar, ya puestos, a hacer los mismos pasos desde XP con el denostado Virtual PC 2007...funcionó a la primera sin tener que renombrar archivos y bastante rápido
