    Books

    This page is the result of parsing the “My Clippings.txt” file that Kindle devices autogenerate with the highlights you make. I created a Nodejs app called MyClippings to Markdown that parses this file and creates a Markdown file for each book, grouping the quotes. I know there are plenty of tools (Clippings.io) and scripts (1,2) doing similar things, but none of them fit my requirements and well I was bored in a rainy quaratined weekend .

    I lost some clippings when I bought a new Kindle. Expect to find books in English and Spanish from very random topics:
