Windows 8.1 - Arreglar error "We weren't able to connect to the store..."



Hola,

Si os habéis decidido por probar la beta de Windows 8.1 puede que os hayais encontrado con el siguiente error al intentar entrar a la Windows Store

“We weren’t able to connect to the store. This might have happened because of a server problem or the network connection timed out. Please wait a few minutes and try again.”

No había forma de acceder a la Store, incluso reiniciando, hice búsquedas porque al parecer es un "known issue" pero lo único que sirvió fué verificar la cuenta de usuario. Dentro de la configuración de cuentas, en vuestra cuenta, saldrá un link (debajo del mail) para verificar vuestro usuario y introducir la contraseña por enésima vez. Una vez hecho esto, desaparece el link y la Windows Store volverá a funcionar.
